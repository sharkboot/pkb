在上一章中，我们介绍了 Model 类和 Tokenizers 类，尤其是如何运用分词器对文本进行预处理。

Transformers 库建立在 Pytorch 框架之上（Tensorflow 的版本功能并不完善），虽然官方宣称使用 Transformers 库并不需要掌握 Pytorch 知识，但是实际上我们还是需要通过 Pytorch 的 DataLoader 类来加载数据、使用 Pytorch 的优化器对模型参数进行调整等等。

因此，本章将介绍 Pytorch 的一些基础概念以及后续可能会使用到的类，让大家可以快速上手使用 Transformers 库建立模型。

1. Pytorch 基础

Pytorch 由 Facebook 人工智能研究院于 2017 年推出，具有强大的 GPU 加速张量计算功能，并且能够自动进行微分计算，从而可以使用基于梯度的方法对模型参数进行优化。截至 2022 年 8 月，PyTorch 已经和 Linux 内核、Kubernetes 等并列成为世界上增长最快的 5 个开源社区之一。现在在 NeurIPS、ICML 等等机器学习顶会中，有超过 80% 研究人员用的都是 PyTorch。

张量 (Tensor) 是深度学习的基础，例如常见的 0 维张量称为标量 (scalar)、1 维张量称为向量 (vector)、2 维张量称为矩阵 (matrix)。Pytorch 本质上就是一个基于张量的数学计算工具包，提供了多种方式创建张量：torch.empty(), torch.rand(), torch.randn(), torch.zeros(), torch.arange(), torch.tensor(), torch.from_numpy() 等。

注意：上面这些方式创建的张量会存储在内存中并使用 CPU 进行计算，如果想要调用 GPU 计算，需要直接在 GPU 中创建张量或者将张量送入到 GPU 中：torch.rand(2, 3).cuda() 或 torch.rand(2, 3).to('cuda')。

张量计算：加减乘除按元素进行计算。还提供了 torch.dot() 计算向量点积、torch.mm() 计算矩阵相乘、三角函数和各种数学函数等。聚合操作如 mean、sum、max、min 可以通过 dim 参数指定维度。torch.cat() 用于拼接张量。

广播机制：即使两个张量形状不同，也可以通过广播机制对其中一个或者同时对两个张量的元素进行复制，使得它们形状相同，然后再执行按元素计算。

索引与切片：与 Python 列表类似，可以对张量的任意维度进行索引或切片。索引值从 0 开始，切片 [m:n] 的范围是从 m 到 n 前一个元素结束。

降维与升维：torch.unsqueeze() 在指定位置插入一维，torch.squeeze() 删除形状为 1 的维度。

自动微分：Pytorch 提供自动计算梯度的功能，只需要执行 tensor.backward() 就会自动通过反向传播算法完成。使用时需要设置 requires_grad=True。

2. 加载数据

Pytorch 提供了 DataLoader 和 Dataset 类专门用于处理数据。

Dataset：所有数据集类都必须继承自 Dataset 或 IterableDataset。映射型数据集必须实现 __getitem__() 和 __len__()；迭代型数据集必须实现 __iter__()。

DataLoader：负责迭代地访问数据集中的样本。常用参数包括 batch_size、shuffle、sampler、collate_fn 等。DataLoader 默认会创建整数索引的采样器。

3. 训练模型

Pytorch 所有的模块（层）都是 nn.Module 的子类。构建模型需要继承 nn.Module，通过 __init__() 初始化层和参数，在 forward() 中定义操作。

优化模型参数：损失函数如 nn.CrossEntropyLoss、nn.MSELoss、nn.NLLLoss 用于度量预测值与答案之间的差异。优化器如 torch.optim.AdamW、SGD 通过在每一个训练阶段减少模型损失来调整参数。

训练循环步骤：
1. optimizer.zero_grad() 重设模型参数的梯度
2. loss.backward() 反向传播计算损失对每一个参数的偏导
3. optimizer.step() 根据梯度调整模型的参数

注意：在预测之前调用 model.eval() 将 dropout 层和 batch normalization 层设置为评估模式。

4. 保存及加载模型

保存模型权重：torch.save(model.state_dict(), 'model_weights.pth')，加载时需要先创建结构相同的模型实例。

保存完整模型：torch.save(model, 'model.pth')，可直接通过 torch.load() 加载。
